# Урок 6. На улицата

## Диалог

— Добър ден, госпожице. Капитан Вълчанов. Моля, документите за проверка.
— Но защо, господин полицай? Аз карам внимателно.
— Сега ще видим. Шофьорската книжка, личната карта и документите на колата, ако обичате.
— Заповядайте.
— Благодаря. Така-а. Значи Вие сте Ружа Шапкарова, от град Велико Търново, улица „Георги Жижков", блок номер 10, вход Г, етаж трети, апартамент 8. Този адрес е в квартал „Бузлуджа", нали?
— Да, там е.
— А къде работите?
— В една банка. Имате ли други въпроси?
— А-а-а... какво ще правите довечера?

## ГРАМАТИКА – ДА-КОНСТРУКЦИЯ / ДА-CONSTRUCTION

| (не) | мога / искам / трябва / обичам / започвам / спирам | да | чета / уча / говоря (глагол в сегашно време / a verb in the present tense) |

- Те искат да излязат.

- Той иска да спи.
- Те могат да работят с компютър.

- Той не трябва да прави това.

## ГРАМАТИКА – ПОВЕЛИТЕЛНО НАКЛОНЕНИЕ / IMPERATIVE

### Положителни форми / Positive forms

**След съгласна / After a consonant:**

| чет-а → чет-и / чет-ете |
| пи-ш-а → пиш-и / пиш-ете |
| гово-р-я → говор-и / говор-ете |

**След гласна / After a vowel:**

| пит-а-м → пит-ай / пит-айте |
| изми-я → изми-й / изми-йте |
| чак-а-м → чак-ай / чак-айте |

### Отрицателни форми / Negative forms
Образуват се само от глаголи от несвършен вид.
*The negative forms have only the imperfective verbs.*

| Спри! | Не спирай! |
| Влез! | Не влизай! |
